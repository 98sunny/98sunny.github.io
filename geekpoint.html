About geekPoint

geekPoint is haven for all the budding devs, tech bloggers, software engineers, data scientists, and anybody who is interested in learning about technology. We create discussion spaces for people in the tech community. Started by a bunch of Humber students with an intend to spread the love of programming with all. We organize code labs, discussions, demos, talks, lightening talks, seminars. All the upcoming events are updated here.
